#YoukiDEX

A powerful desktop experience layer for Android Bring a full desktop environment to your Android device — dock, hot corners, freeform windows, and more.

✨ Features Floating Dock — A persistent, customizable app dock rendered as a system overlay using Android's Accessibility Service. Hot Corners — Trigger actions by moving your cursor or finger to any corner of the screen. Freeform Window Support — Launch and manage apps in resizable freeform windows for a true desktop feel. App Drawer — A full-featured launcher with fast app search and sorting. Deep Shortcuts — Long-press app icons to access app-specific shortcuts, just like a real desktop right-click menu. Icon Pack Support — Compatible with third-party icon packs for full visual customization. Notification Panel — An integrated notification viewer accessible from the dock. System Tray — Live indicators for battery, Wi-Fi, Bluetooth, and sound status. Keyboard Shortcuts — Hardware keyboard support for power users. Multi-Display Support — Works across multiple connected displays. Appearance Theming — Customize colors, transparency, and layout of the dock and UI. Sound Events — Audio feedback for dock and launcher interactions. 📸 Screenshots (Add your screenshots here) 🔧 Requirements Requirement Value Minimum Android 7.0 (API 24) Target Android 14 (API 34) Language Kotlin 1.9.20 Build Tool Gradle IDE Android Studio Hedgehog+ 🚀 Getting Started

Clone the repository git clone https://github.com/mrYouki/YoukiDex-.git cd mrYouki
Open in Android Studio Open the project folder in Android Studio (Hedgehog or newer). Wait for Gradle sync to finish.
Build & Run ./gradlew assembleDebug Or use the Run button in Android Studio to deploy directly to a connected device. 🔐 Permissions YoukiDEX requires several elevated permissions to function as a desktop environment: Permission Purpose SYSTEM_ALERT_WINDOW Draw the dock and overlays on top of other apps ACCESSIBILITY_SERVICE Detect and respond to system-level UI events PACKAGE_USAGE_STATS Show recent and running apps WRITE_SECURE_SETTINGS Apply system-level customizations QUERY_ALL_PACKAGES Enumerate installed apps for the app drawer NOTIFICATION_LISTENER Display notifications in the dock panel ⚠️ Some permissions require ADB or a rooted device to grant (e.g., WRITE_SECURE_SETTINGS). Grant via ADB: adb shell pm grant com.youki.dex android.permission.WRITE_SECURE_SETTINGS
